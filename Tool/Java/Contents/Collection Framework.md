# Collection Framework

**컬렉션 프레임워크**란 다수의 데이터를 저장하는 클래스들을 표준화한 설계를 의미한다. 컬렉션 프레임워크는 다양한 클래스들을 제공하고, 인터페이스와 다향성을 이용한 객체지향적 설계를 통해 표준화되어 있기 때문에 재사용성 뫂은 코드를 작성할 수 있다.
